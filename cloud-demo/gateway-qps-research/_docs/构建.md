# 构建

## 设定版本

```bash
# 用 cmd 运行

# JDK 版本
set JAVA_HOME=D:\Install\Java\JDK\jdk-17
set PATH=%JAVA_HOME%\bin;%PATH%

# Gradle 版本
set Gradle=D:\Install\Java\Builder\gradle-8.8
set PATH=%Gradle%\bin;%PATH%

gradle -v
```

## 查看依赖

```bash
# 查看 spring-gateway 模块所有 Classpath 依赖
gradle :spring-gateway:dependencies > deps.txt

# 只查看 spring-gateway 模块 compileClasspath 依赖
gradle :spring-gateway:dependencies --configuration compileClasspath > deps.txt
```

## 清理

```bash
gradle clean
```

## 构建

```bash
gradle build -x test
```
