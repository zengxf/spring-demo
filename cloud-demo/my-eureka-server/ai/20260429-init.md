请帮我初始化一个 Spring Cloud Eureka Server 项目，具体要求如下：

项目基础：使用 Java 21 和 Maven 构建，Spring Boot 版本建议使用 3.2，Spring Cloud 版本使用对应的稳定版。

依赖配置：添加 spring-cloud-starter-netflix-eureka-server 依赖。

启动类：在主类上添加 @EnableEurekaServer 注解。

配置文件：创建 application.yml，设置端口为 8761，并配置 Eureka 基础参数：

client.register-with-eureka: false

client.fetch-registry: false

设置服务名称为 my-eureka-server。

目录结构：请生成标准的 Maven 目录结构，并包含基础的 .gitignore。
