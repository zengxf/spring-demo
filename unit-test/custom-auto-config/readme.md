## 总结
- `ApplicationConetxtRunner` 适用于非 Web 应用程序
- `WebApplicationContextRunner` 用于基于 Servlet 的 Web 应用程序
- `ReactiveWebApplicationContextRunner` 用于响应式 Web 应用程序
